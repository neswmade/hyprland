# nesw hyprland
## install

```bash
mv ~/.config/hypr ~/.config/hypr.bak
git clone https://github.com/neswmade/hyprland.git ~/.config/hypr
```

## dependencies
 - ghostty (terminal)
 - helium-browser
 - app2unit
 - wl-clipboard
 - cliphist
 - gnome-keyring
 - polkit-gnome
 - geoclue
 - gammastep
 - bluez-utils
 - Bibata-Modern-Ice (cursor)
 - trash-cli
 - qt5-wayland
 - qt6-wayland
 - pipewire
 - pipewire-pulse
 - wireplumber
 - xdg-desktop-portal-hyprland
### optional
 - nvidia drivers
